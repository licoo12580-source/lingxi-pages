🦞 灵犀 · Lingxi

酒店房间空间感知系统 · v0.3
一块板子装进客房，感知有没有人、温度、门窗状态，自动控空调节省10-20%电费。
同时识别语音指令，提供场景化服务。空间知道你在做什么。

📦 模块说明

固件模块（C语言，ESP-IDF框架）
模块功能状态说明
rd2450.c RD2450雷达驱动 ✅ 完成 UART协议解析，目标坐标/速度提取，多目标跟踪
ms24.c 觅感MS24雷达驱动 ✅ 完成 Signed Magnitude编码解析，坐标/能量/运动检测
ar780.c AR780雷达驱动 ✅ 完成 微动感知，低功耗唤醒
fusion.c 传感器融合 ✅ 完成 状态机：无人→有人→微动→稳定→睡眠→离床检测
energy.c 节能控制 ✅ 完成 三模式：深度节能/舒适节能/性能模式，空调联动
mqtt.c MQTT通信 ✅ 完成 Air780E AT指令驱动，JSON消息上报/接收
display.c ST7789屏幕驱动 ✅ 完成 2.8寸SPI屏，中文显示，系统状态
touch.c FT6336U触摸 ✅ 完成 I2C接口，电容触摸，手势识别
voice.c 语音识别 🔧 原型 ES7210 ADC + INMP441×3，离线识别"太冷了""需要打扫"等
main.c 主程序 ✅ 完成 系统初始化、任务调度、看门狗

后端服务（服务器端）
服务技术栈状态说明
EMQX Broker EMQX 5.x ✅ 运行 MQTT消息中间件，端口1883
Gateway Vue + Node ✅ 运行 Web网关，设备管理，状态面板
后端API Node.js ✅ 运行 REST API，端口3001
ASR服务 Python ✅ 运行 离线语音识别服务
Hermes Agent Python ✅ 运行 AI运维机器人，群消息回复
