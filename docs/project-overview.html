给虾条 · 当前方案与分工
项目在做什么
酒店房间空间感知系统：一块板子装进客房，知道有没有人、温度多少，自动控空调节能10-20%。同时感知门窗状态、语音指令、设备健康。
核心卖点：空间感知能力。不是单个功能，是"空间知道你在做什么"。
当前硬件方案（研究确定版）
ESP32-S3 N16R8 主控
RD2450 雷达1 + AR780 雷达2（双雷达融合）
ES7210 音频ADC + INMP441×3（离线语音指令）
Air780E 4G模块 + 2.8寸IPS屏
SHT30 温湿度 + 继电器+红外
BOM约¥128（标准版）
跟之前方案的变化
• 语音路径改掉了：之前INMP441直连S3内置ADC（SNR~50dB废的）→ 现在加ES7210（102dB SNR，可用）
• 屏换了：SH1106 OLED小屏 → 2.8寸IPS彩屏
• 雷达升级：单LD2450 → RD2450+AR780双雷达融合
• 温湿度升级：DHT11 → SHT30
• 成本基本没变，能力翻倍
