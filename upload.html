    手动提交指引

    如果GitHub API方式不方便，可以按以下步骤手动提交：

    1. 克隆仓库：git clone https://github.com/licoo12580-source/lingxi-pages.git
    2. 将文件放入对应目录（firmware/ 或 docs/）
    3. 提交并推送：git add . && git commit -m "描述" && git push
    4. 更新 firmware.html（如果是固件，需要添加版本条目到列表）

    📋 固件条目模板

    复制以下JSON模板，替换内容后添加到 firmware.html 的 versions 数组中：

    {
  ver: "vXXXX",
  date: "2026-06-20",
  status: "latest",
  tags: ["标签1", "标签2"],
  file: "vXXXX.bin",
  sha256: "sha256值",
  size: "XXXKB",
  driver: "ILI9488",
  desc: "版本描述",
  changelog: "🔧 改动1\\n🔧 改动2",
  hw: "HW-678A N16R8, 4寸屏"
}
