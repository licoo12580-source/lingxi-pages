⏱ 目标：从拆箱到上线 ≤15分钟

📦 灵犀安装指南

适合：酒店前台 / 保洁 / 老板 · 不需要专业技能
1

📋 装前检查

  • 空调是分体式（壁挂/柜机），不是中央空调
  • 空调遥控器能正常开关
  • 房间有电源插座
  • 已确认房间号
❌ 中央空调暂不支持，等V1.5网关方案
2

📦 拆箱 · 检查配件

📡
灵犀主机 ×1
🔌
电源适配器 ×1
🔋
USB-C线 ×1
🏷️
快速入门卡 ×1
⚠️ 保留包装盒。如需寄回翻新/维修，用原包装
3

⚡ 通电

1. 插USB-C线到设备

2. 插电源适配器到插座

3. ⏳ 等待指示灯变化：

红灯闪烁 → 启动中
蓝灯慢闪 → 联网中
绿灯常亮 → ✅ 在线
🟢 绿灯常亮 = 联网成功！
🔴 红灯快闪 = 联网失败（见底部排查）
4

🔗 确认在线

打开手机扫设备上的二维码 → 显示"设备在线"即可

或让序章在后台上确认

✅ 在线 → 进入下一步
5

🏷️ 绑定房间

告诉序章：

  • 设备ID（设备标签上）
  • 房间号（如：301）
  • 空调品牌（格力/美的等）

序章在后台绑定后，会测试空调开关

✅ 空调能开能关 → 绑定完成
⚠️ 不能控制 → 试品牌自动搜索
6

✅ 验收

  • 绿灯常亮（在线）
  • 空调可远程开关
  • 雷达能检测有人/无人
  • 温度读数正常
🎉 装完！15分钟搞定一间

📐 安装位置建议

  • ✅ 高度1.5-2米（视线高度）
  • ✅ 面向房间中央
  • ✅ 靠近电源插座
  • ❌ 不要放空调正下方
  • ❌ 不要放金属柜内（4G信号屏蔽）
  • ❌ 不要放窗边（阳光直射）

🏢 一层楼批量装（10间）

08:00 全部10台拆箱插电
08:15 10台全绿灯 → 确认在线
08:20 开始绑定（3分钟/间）
08:50 10间全绑定
08:55 每间验收（1分钟/间）
09:05 ✅ 全部完
约6.5分钟/间 · 批量更高效

🔧 故障排查

红灯快闪（联网失败）：

① 检查天线是否拧紧
② 确认位置有4G信号（手机测）
③ 重启设备（拔电源重插）
④ 换离窗户近的位置
⑤ 仍不行→联系序章换卡

空调不能控制（红外失败）：

① 红外窗对准空调方向
② 确认品牌选对了
③ 试品牌自动搜索
④ 仍不行→用原装遥控器学习

雷达不准确：

① 确认高度≥1.5米
② 前方无障碍物
③ 启动后等2分钟自校准
④ 检测范围约5×5米
