# 灵犀 · 安装 SOP（标准作业流程） > 版本: v1.0 · 更新: 2026-06-25 > 适用对象: 酒店前台/保洁/业主自行安装 > 目标: 5分钟完成，无需专业工具，零上门 --- ## 一、开箱检查 收到灵犀设备后，确认包装内含： | 物品 | 数量 | 说明 | |:----|:----|:----| | 灵犀主机 | 1台 | 含4G模块+雷达+红外发射 | | USB电源线 | 1根 | Type-C, 1.5m | | 充电头 | 1个 | 5V/2A | | 壁挂底座（选配） | 1个 | 3M胶贴，无需打孔 | | 快速入门卡 | 1张 | 二维码扫了就看视频 | --- ## 二、安装位置选择 ``` 空调室内机 ┌──────────────┐ │ │ │ ←←← 红外信号 →→→ ← 灵犀放这里，红外正对空调 │ ┌──────┐ │ ← 距离 < 5米 │ │ 灵犀 │ │ │ └──────┘ │ │ │ └──────────────┘ 房间内部 ┌──────────────┐ │ │ │ 雷达覆盖 → │ ← 120°扇形，探测有人/无人 │ 门 │ └──────────────┘ ``` ### 位置要点 1. **红外直射空调** — 设备红外发射窗口对准空调接收窗（<5米） 2. **雷达无遮挡** — 前方不要有屏风/窗帘遮挡探测角度 3. **远离金属** — 不要紧贴金属表面（影响4G天线信号） 4. **高度适中** — 桌面/床头柜/墙面均可，1~2米高度最佳 5. **有电源** — 附近1.5米内有USB供电口 > ❌ 不要放空调顶部（雷达被遮挡） > ❌ 不要放金属货架内部（信号屏蔽） > ❌ 不要放浴室/窗台（潮湿+阳光直射） --- ## 三、通电开机 1. 插上Type-C电源线 2. **红灯常亮** → 正在启动（约10秒） 3. **绿灯闪烁** → 启动完成，等待联网 4. **绿灯常亮** → 已联网，正常运行 ### 指示灯说明 | 状态 | 灯色 | 含义 | |:----|:----|:----| | 常亮 | 红灯 | 启动中 / 设备故障 | | 闪烁 | 绿灯 | 联网中（正常情况下15秒内变常亮） | | 常亮 | 绿灯 | ✅ 正常运行 | | 闪烁 | 蓝灯 | 正在OTA升级（不要断电！） | | 常亮 | 红灯+绿灯 | 设备异常，联系客服 | --- ## 四、绑定房间 ### 方法A：微信扫码（推荐） 1. 用微信扫设备底部二维码 2. 关注「灵犀空间」公众号 3. 点击「添加设备」→ 扫码绑定 4. 输入房间名（如「101大床房」） 5. 选择空调品牌 → 系统自动匹配码库 6. ✅ 绑定成功，开始检测 ### 方法B：管理后台 1. 打开酒店管理后台（网址见快速入门卡） 2. 用管理员账号登录 3. 点击「添加设备」→ 扫码 4. 进入设备详情的「红外学习」→ 点击测试 5. 如果自动码库匹配不上 → 用原遥控器对准灵犀按一次开机键 --- ## 五、功能验证 绑定后做以下测试，至少花2分钟验证： - [ ] 开/关 — 用灵犀控制空调开关，看空调是否响应 - [ ] 调温 — 从26℃调到22℃，看空调是否变冷 - [ ] 模式切换 — 制冷→送风→制热（视季节） - [ ] 风速 — 高→中→低 - [ ] 有人检测 — 站在雷达前晃手，30秒内应看到状态变「有人」 - [ ] 无人检测 — 离开房间，3~5分钟变「无人」，空调自动调温 ### 遥控器学习模式 如果自动码库控制不了空调： 1. 在管理后台进入「红外学习」 2. 点击「开始学习」 3. 用原遥控器对准灵犀，按一次「开机键」 4. 重复步骤2-3，分别学习：关机、调温（26℃、24℃、22℃）、风速、模式 5. 学到6个以上按键后点击「完成」 6. ✅ 学习成功，以后可直接控制 --- ## 六、常见问题 | 问题 | 原因 | 解决 | |:----|:----|:----| | 绿灯一直闪 | 4G信号弱 | 换位置 / 加外接天线 | | 空调不响应 | 红外没对准 | 调整角度，确保红外窗口正对空调 | | 控制但没有反馈 | 码库不完全匹配 | 用学习模式重新学习 | | 雷达一直显示无人 | 角度不对 | 调整朝向，排除遮挡物 | | 检测不准（有人显示无人） | 人在雷达边缘 | 重新安装位置，让雷达正对活动区域 | | 频繁误报 | 窗帘/风扇被识别为有人 | 调整灵敏度（管理后台->设备->雷达设置） | --- ## 七、维护 - 每月清理一次设备表面灰尘（干布擦拭） - 无需更换电池（直接USB供电） - 设备自动OTA升级（凌晨3-5点，不影响运行） - 如设备红灯常亮 → 拔电重插即可 - 退房 → 在后台解绑设备 → 新客人入住自动重新绑定 --- ## 八、不支持的情况 | 情形 | 说明 | 替代方案 | |:----|:----|:----| | 中央空调（多联机/风管机） | 红外不可控 | 温控器模块（另购） | | 窗式空调 | 多为机械旋钮 | 需改电动执行器 | | 老旧空调（15年+） | 红外协议不标准 | 学习模式可试，不保证 | | 蓝牙/APP-only控制 | 无红外信号 | 暂不支持 | --- > 📌 **安装后请务必做一件事**：在管理后台确认设备在线状态，状态为「绿色在线」才算完成。
