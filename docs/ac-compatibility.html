📋 快速自检表

对照你的空调遥控器，逐项确认：

  • 空调是 分体式壁挂/柜机（非中央空调/风管机）
  • 遥控器是 红外 信号（非蓝牙/RF无线）
  • 遥控器有 开关、模式、温度、风速 按键
  • 空调品牌在下方 已支持品牌清单 中
  • 遥控器型号标签可看清（机身或电池仓内）
  • 空调附近 有220V插座（灵犀需供电）
  • 房间有 WiFi或4G信号（物联网卡需信号）
  • 空调 能正常制冷/制热（设备只优化不维修）

📦 安装前准备清单

灵犀设备到货后，安装前确认以下事项：

  1. 设备已充电/通电（红灯亮起）
  2. 手机已连接与灵犀相同的网络环境
  3. 安装位置在空调可视范围内（红外直射，<5米）
  4. 安装位置不能遮挡雷达探测角度
  5. 设备不要紧贴金属表面（影响4G天线）
  6. 测试遥控器是否正常（装设备前先手动开/关一次空调）
  7. 记录房间名称/编号（后续绑定用）

⚠️ 不兼容怎么办？

情景1：品牌不在列表 → 拍照遥控器发给灵犀团队，3个工作日内录入码库
情景2：中央空调/风管机 → 需额外加装温控器模块，联系灵犀工程部
情景3：红外但控制不了 → 设备自带学习模式，对准原遥控按一下即可学习
情景4：无插座 → 提供USB供电方案（需另购电源延长线）
情景5：4G信号弱 → 可外接天线增强信号
